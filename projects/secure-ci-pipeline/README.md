# Secure CI/CD Pipeline

## Overview
This project demonstrates a **secure CI/CD pipeline** that integrates:
* Static Analysis (SAST) - SonarQube  
* Dependency Scanning - Snyk  
* Container Security - Trivy  
* Infrastructure Scanning - Checkov  

## Tech Stack
- GitHub Actions
- Terraform
- AWS
- Docker, Kubernetes
